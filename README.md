# Trainerr

## Tech Stack

> React, Vite, TypeScript, TailwindCSS + daisyUI, ESLint + Prettier, React Router, Redux, i18next

## Setup

### Install Dependencies

`npm install`

### Run Dev mode

`npm run dev`
